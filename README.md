# Matty’s Site

This is my personal website built with HTML and CSS.  
It includes an About Me section, a photo gallery, and a small blog.

## Live Site
https://mcarullo-tech.github.io/mattys-site/

## Structure
- `index.html` — main page  
- `styles.css` — site styling  
- `images/` — pictures and icons  
- `posts/` — blog posts
